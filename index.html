Currency Conversion API

Endpoint

https://jeapis.netlify.app/.netlify/functions/currency?from=<currency_code>&to=<currency_code>&amount=<int>

Parameters

  • from: Currency code to convert from.
  • to: Currency code to convert to.
  • amount: Amount of currency to convert.

Example Request

https://jeapis.netlify.app/.netlify/functions/currency?from=USD&to=EUR&amount=100

Response

{
    "value": "<value>",
    "rates": {
        "<currency_code>": "<value>"
    }
}

The API response includes the converted value and the exchange rates for the given currencies.
